⚡ EV Data Analysis Dashboard (Tableau)
📌 Project Overview

This project analyzes Electric Vehicle (EV) adoption trends using Tableau.
It provides insights into vehicle types, manufacturers, state-wise distribution, CAFV eligibility, and model year growth trends.
